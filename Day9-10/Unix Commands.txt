DF command -- displays the amount of space available on the file system 
	      containing each file name argument

	like --block-size=1K

        print sizes in powers of 1024 (e.g., 1023M)

FREE command -- display amount of free and used memory in the system

free -b
	Display the amount of memory in bytes.

free -k
	Display the amount of memory in kibibytes. This is the default.

free -m
	Display the amount of memory in mebibytes.

free -g
	Display the amount of memory in gibibytes.

TOP Command -- display Linux processes

top -n 1 -b | head -15
	-n 1 runs top for one iteration.
	-b runs it in batch mode (so it prints to stdout).
	head -15 shows the first 15 lines (header + ~5 processes).

top -b -n 1 | sed -n '8,12p'    
	This prints lines 8–12, which correspond to the first 5 processes

ps -eo pid,comm,%cpu,%mem --sort=-%cpu | head -6
	This shows the top 5 CPU-consuming processes (plus the header)

PS command -- displays information about a selection of the active processes.

       To see every process on the system using standard syntax:
          ps -e
          ps -ef
          ps -eF
          ps -ely

       To see every process on the system using BSD syntax:
          ps ax
          ps axu

GREP command -- searches  for  PATTERNS  in each FILE
	
	ps -ef | grep amazon
------------------------------------------------------------------------------
AWK command -- pattern scanning and processing language
	
	ps -ef | grep amazon | awk -F" " '{print $2}'
	this will display the process ID of amazon processes
